Project Hexarover

A Modular Environmental Sensing Platform

Welcome

Hexarover is a modular robotic platform designed for environmental sensing and exploration. At its core, it combines Arduino compatible hardware integrated with versatile software architecture to create a highly adaptable rover system.

This documentation covers the design, assembly, and operation of the Hexarover platform.
