Project Hexarover

A Modular Environmental Sensing Platform

Overview

Hexraover is a modular environmental sensing rover with these key technical components:

  • Core Platform: Arduino Uno controlling 4 TT motors via L298N motor drivers, powered by dual 18650 batteries (5000mAh) with voltage regulation through a DC-DC buck converter
  • Communication System: ESP32-CAM modules providing WiFi connectivity, video streaming, and REST API endpoints for remote control and data access
  • Distributed Computing Architecture: Each sensor pack will have its own Arduino Nano microcontroller, communicating with the main Arduino via I2C protocol
  • Modular Hardware Design: Custom 3D printed chassis with a Universal Mounting Bracket (UMB) system for tool-free swapping of sensor modules
  • Sensor Categories:
    • Atmospheric monitoring (temperature, humidity, pressure, etc.)
    • Terrain analysis (soil moisture, vibration, inclination)
    • Navigation awareness (ultrasonic, infrared, obstacle detection)
  • Software Framework: Web interface for control and data visualization, with a REST API enabling configuration of which sensors are connected to which pins
  • Mechanical Features: Hybrid PLA/TPU wheel design with 3D printed cores and flexible TPU tires for better traction

The project emphasizes modularity throughout - both physically through the universal mounting system and electronically through the distributed microcontroller approach. This allows for customization and incremental development while maintaining a clean separation of concerns in the code.

Hexarover's Story

Hexarover began as a small idea that evolved into something much larger and much more meaningful. My goal for this project is to create an accessible, reasonably affordable, open platform for exploration, fun, and learning. Seeing Hexarover show up in someone's school robotics program or STEM learning camp, or simply hearing of other people enjoying the project would be a dream come true for a very special reason.

This rover is the type of project my father and I would have undertaken together when I was a young teen. He was always making, repairing, building, and creating and he had a fondness for electronics and technology. He instilled this curiosity and creativity in me and I credit it for much joy and the success I've had in my life.

I'm a woman who was born blind in the 70s and grew up poor. I doubt anyone expected me to go into a STEM field. But my father never let cultural expectations or my own supposed limitations get in the way of helping me pursue my interests.

This project is a tribute to that spirit and that dedication.
