Project Hexarover

Appendix A: Parts List

This list attempts to be thorough and complete. However, there may be some small items missing. Please read build instructions carefully for a full understanding of necessary parts.

Sensor Pack Note: The link to the sensor pack takes you to a listing with multiple options. Please use the 22 sensor pack, which should provide you with most, if not all of the sensors on this list. This will save you a lot of money over buying sensors individually. That said, the great part about Hexarover is that if this specific pack is unavailable, you can always build your own modules!

Item Quantities: Many of these components (aside from the sensor pack) ship with multiple quantities of a component, often times at least two or three. This can significantly lower the cost per rover if you're looking to build multiple rovers as part of a group or class project.

Tools & Materials

Tool Note
3D Printer Required - printing chassis & mounting brackets
CA Glue Required - to affix some mounts to chassis
Sandpaper Required - 220g works well
Soldering Iron & Solder Required - 0.8-1mm solder works well
Wire Stripper Strongly Recommended
Small Screwdriver Likely Required - battery box mount, etc.
Hobby Knife Strongly Recommended
Digital Calipers or Milimeter Ruler Optional - necessary for custom modules
Painter's Tape Optional - for temporary mounting & print bed adhesion
Lighter Optional - for small pieces of heat shrink
Multimeter Optional, but recommended
Magnifier Optional - useful for inspecting solder joints on small components
Peel & Stick Labels Recommended - for labling JST wires
